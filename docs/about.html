About

BaseWeb is a front-end development framework written in SCSS. It was created from a need to reduce the amount of time writing CSS while giving control over CSS output to the author.

BaseWeb establishes a methodology of organizing code to be easily maintained, updated and internally consistent while keeping the final CSS output as clean and minimal as possible.

Who made BaseWeb?

BaseWeb was created and is maintained by Sebastian Nitu, a User Interface designer and developer working at Avelient.
